# QAffeine

Vítejte v projektu **QAffeine**! Jedná se o webový web týmu vytvořený jako součást Czechitas Git cvičení, který představuje členy týmu a jejich profily.

## 📋 O projektu

QAffeine je jednoduchý HTML a CSS projekt, který seznamuje s členy týmu a demonstruje kolaborativní vývoj webových stránek pomocí správy verzí Git.

## 🎨 Vlastnosti

- Čistá, sémantická HTML struktura
- Responzivní CSS styling
- Sekce profilů členů týmu s jednotlivými ID pro snadný odkaz

## 🚀 Co je za úkol

1. Naklonujte si repozitář Vašeho týmu

2. Vytvořte lokální feature branch a v této udělejte úpravy, které commitnete

3. Nasdílejte **změny** ve Vaší feature branchi s ostatními přes GitHub

4. Vytvořte Pull Request

5. Mergujte Vaše změny do hlavní větve main

## ✏️ Jak na změny

Úkolem každé z vás bude upravit sekci o Vás:

1. **Najděte si svou sekci** v `index.html`

2. **Upravte své jméno** – v tagu `<h2>` můžete text pozměnit nebo doplnit

3. **Napište o sobě** – nahraďte zástupný text (`... zde napište něco o sobě ...`) jednou nebo dvěma libovolnými větami o sobě

4. **Vložte si obrázek** – přidejte si do své sekce fotku či obrázek podle svého výběru (např. `<img src="path/to/image.jpg" alt="vaše jméno">`)

5. **Formátujte svou sekci** (volitelně):
   - Změňte styl nebo barvu písma
   - Upravte barvu pozadí
   - Přidejte si vlastní CSS pravidla do `mycss.css` zaměřená na vaše ID sekce

6. **Commitněte a pushněte** své změny do Gitu
